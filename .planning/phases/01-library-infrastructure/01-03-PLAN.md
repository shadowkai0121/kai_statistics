  component: PyPI publishing identity

  category: information_disclosure
  component: contributor and installation docs
  disposition: mitigate
  mitigation_plan: Document only non-secret setup steps, point notebook users to `pip install kai_statistics`, and avoid instructing maintainers to commit tokens or account secrets.
</stride_register>
